Quick_Start_Guide.sh
01

解压 获取文件包，确保 .exe 与 .pck 在同一目录下。

02

pip install 运行前请确保已安装所需环境：
mediapipe opencv-python

03

Python 运行 hand_controller.py 开启手势捕捉。

04

开始游戏 直接双击运行 CyberSmash.exe 开启赛博城市跑酷之旅！

⚠️ 注意：手势控制器必须先于游戏开启并在游戏运行中保持开启。
